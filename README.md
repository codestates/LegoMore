- 갖고 싶었지만 너무 비싸서 가질 수 없었던 레고를 구입할 수 있어요
- 내가 찾아보고자 하는 레고만 필터링하여 볼 수 있어요
- 마음에 들었던 레고를 찜 할 수 있어요
- 마이페이지에서 내가 찜한 레고만 모아 볼 수 있어요
- 연락하기를 통해 대화를 나눌 수 있어요
- 이 가격이면 충분할 것 같다고 생각이 들 때에는 쿨 거래로 즉시 구매를 할 수 있어요
